# Cocol
**Install Cocol**
```bash
$ brew install cocol
```

**Create a new project**
```bash
$ cocol init my_microchain

Welcome to cocol

Choose a consensus mechanism

1. Proof of Work
2. Proof of Stake
3. PoW & PoS Hybrid
```

**Startup network**
```bash
$ cocol start my_microchain

Spawning nodes (200) ....... [OK]
Apply magic ................ [OK]
Start Dasboard ............. [OK]

Ready! Check http://localhost:8989
```
