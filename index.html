We aim for developer happiness with a quick and straightforward setup.

You can choose from a variety of community submitted modules to create your own blockchain and immediately start your new project.
Interact with your new blockchain via a dashboard and see e.g. how transaction move along the network until consensus is reached.

1. Quick Setup

# Install Cocol
$ brew install cocol

# Create a new project
$ cocol init my_chain

Welcome to Cocol

Choose a consensus mechanism

1) Proof of Work
2) Proof of Stake
3) Hybrid PoW + PoS
